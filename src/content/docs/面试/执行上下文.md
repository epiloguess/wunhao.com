### GEC

> 每当 JavaScript 引擎收到脚本文件时，它首先创建一个默认的执行上下文，称为  `Global Execution Context (GEC)` 。
> GEC 是基本/默认执行上下文，所有不在函数内部的 JavaScript 代码都会在其中执行。

### FEC

> 每当调用函数时，JavaScript 引擎都会在 GEC 中创建一种不同类型的执行上下文（称为函数执行上下文 (FEC)），以评估和执行该函数中的代码。
