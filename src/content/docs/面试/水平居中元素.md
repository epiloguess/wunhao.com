### inline 行内元素

`text-align: center` 只适用于块级元素

对于 `span` 等行内元素，可以使父元素居中从而使行内元素居中

### block 块级元素

对于 `h1` `p` 等块级元素，可以使用`text-align: center`

其他的

```css
margin: 0 auto
```

> 必须有宽度

### float 浮动元素

```css
positon: relative;
left:50%;
transform: tanslateX(-50%);
```
