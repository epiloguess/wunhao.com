### 对调两个变量

#### 使用 temp 变量

#### 加减法

```js
let a = 3,b = 5;
a = a + b;
b = a - b;
a = a - b;
```

#### 数组法

```js
let a=3,b=5;
a=[a,b];
b=a[0];
a=a[1];
```
