### 超出宽度的文本替换为省略号

#### 单行文本处理

```css
/* 设置固定宽度 */
width:100px;
/* 强制不换行 */
white-space:nowarp;
/* 隐藏溢出内容 */
overflow:hidden;
/* 省略内容 */
text-overflow:ellipsis;
```
