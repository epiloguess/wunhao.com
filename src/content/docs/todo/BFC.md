#### 如何触发 BFC

小结触发`BFC`方法：

1. `float`属性为`left`/`right`
2. `overflow`为`hidden`/`scroll`/`auto`
3. `position`为`absolute`/`fixed`
4. `display`为`inline-block`/`table-cell`/`table-caption`
